Teryx Panel
## Overview
Installation
Clone the repository or download: `git clone https://github.com/teryxlabs/v4panel`
`curl -sL https://deb.nodesource.com/setup_23.x | sudo bash -`

`apt-get install nodejs git`

go to panel directory: `cd v4panel`

Install some importent: `apt install zip -y && unzip panel.zip`

Install dependencies: `npm install && npm run seed && npm run createUser`

Start the Panel: `node .` # or use `pm2` to keep it online
